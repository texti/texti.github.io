## Text Formatting

### :white_check_mark: Italic / Emphasis

```
''italic text'' or
''emphasized text''
```

**Additions**. Alternative Syntax

```
_italic text_ or
*emphasized text*
```


### :white_check_mark:  Bold / Strong Emphasis

```
''bold text'' or
''strongly emphasized text''
```

**Additions**. Alternative Syntax

```
__bold text__ or
**strongly emphasized text**
```
